# old_tools について

このディレクトリには、傭兵ログトラッカーの過去バージョン（ver117〜ver155）を
**アーカイブとしてそのまま**保存しています。`version_selector.js` から
iframe経由で参照され、ユーザーが過去バージョンの見た目・挙動を確認できるようにするための
ものです。

## 重要：修正対象外です

- 各バージョンは完全に独立したファイルであり、共通のロジックを持ちません。
- 現行版（`tools/expmercenary.js`）で見つかったバグ修正・計算式の誤り修正などは、
  **このディレクトリのファイルには反映されません**。
- 新規に手を加える予定もありません。あくまで「当時こうだった」という参考・アーカイブ目的です。

## 例外的にlocalStorageを使用しているバージョン

- `ver141.html`は、ダークモード切り替えを`localStorage`の`darkMode`キーで
  自己完結的に保存・復元しています。
- 本来、ver1.2.6・ver1.3.0以外のバージョンはブログヘッダー側（index.html/親ページ）の
  ダークモード制御に追従する想定でしたが、ver1.4.1だけダークモードトグル実装時に
  ツール内へ直接組み込んでしまった一回限りの実装です。「あるべき設計」ではなく
  「当時そう実装してしまった」履歴的な経緯によるものです。
- そのためこのファイルだけ、`iframe`をサンドボックス化する際に`allow-same-origin`を
  付与しないと、この自己完結型のダークモードトグルが（`localStorage`アクセス不可により）
  動作しなくなります。アーカイブとして凍結し中身を修正しない方針である以上、
  `version_selector.js`側で`allow-same-origin`を残す形で対応しています。
- 新しいバージョンをここに追加する場合、`localStorage`等の同一オリジン権限が
  必要かどうかを確認し、`version_selector.js`の`NEEDS_SAME_ORIGIN`配列を
  更新してください。
